# CAST Website Spec

Website for CAST (Connected Autonomous Safe Technologies) lab at Texas A&M University.

- Re-use common code (e.g. header, footer, etc.)
  - Need to determine if we should use a framework (e.g. React, Vue, etc.)
- Use Canva as reference for layout
- Include effects (e.g. hover, click, etc.)
  - For example, on the projects page, when the user hovers over a project, the background photo should slightly enlarge and gray out while text appears over it
- The website should feel modern and professional, but also flashy and engaging
- The font should Oswald, because that is one of the official university fonts
- The color scheme should be these official university colors:
    Primary Red: #500000
    Secondary Red:  #732f2f 
    White: #FFFFFF
    Light Gray: #eaeaea
- Accessibility
    - Should be easy to navigate
    - Should resize nicely
    - Should look good on both desktop and mobile

- Maintainability
  - Should be easy to add a new project
  - Should be easy to add a new team member
  - Should be easy to add a new publication
  - Should be easy to add a new award
